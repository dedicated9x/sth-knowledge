Pendrive:
1. Utworz folder glowny (nazwij go np. "root")

Router:
1. AiDisk (dojdź do końca) (! limitless i bez DDNS)
2. Administration -> Change Mode -> Access point

PC (klient):
1. Wejscie do "network location"

A. "Run..." -> "\\192.168.1.1" (! trzeba będzie zainstalować addon do W10)
B. This PC -> Network (lewy panel) -> Ikona komputera z nazwą routera
C. This PC -> Computer (Tab) -> ftp://192.168.33.6

WAZNE:
W opcjach A i B moza kliknac prawym na folderze i dac "Map as drive" 99% skutecznosci, ale ... szansa, ze "zadzieje sie"
A lub B jest niewielka. Opcja C sluzy tylko i wylacznie "odblokowaniu punktu 2".

2. Dodanie "network drive"
This PC -> Computer (Tab) -> Map network drive -> \\192.168.33.6\root

3. Reconnect przy reboocie -> https://www.youtube.com/watch?v=q8KVJdgewNk
